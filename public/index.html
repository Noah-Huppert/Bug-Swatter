Bug Swatter
Personalized bug tracker management
About
Bug Swatter is a chrome extension that allows for better, more personalized bug tracker management. Bug Swatter allows you to put custom labels on bug reports. These labels allow you to more efficiently moderate bug trackers by only sorting bug reports once.
Installation
Bug Swatter is a chrome extension so chrome must be installed.

Installation:
- Download the latest version of Bug Swatter
- Open chrome settings
- Navigate to the extensions tab
- Drag the Bug Swatter download onto the page
